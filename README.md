# Gains

A SwiftUI application for tracking fitness and gains.

## Requirements

- Xcode 15.0 or later
- iOS 17.0+ / macOS 14.0+ / watchOS 10.0+ / tvOS 17.0+

## Setup

1. Open `Gains.xcodeproj` in Xcode
2. Build and run the project

## Project Structure

- `Gains/` - Main app files
- `Views/` - SwiftUI views
- `Theme/` - App theming and colors
- `Models/` - Data models
- `ViewModels/` - View models
- `Services/` - Service layer
